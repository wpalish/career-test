# Кем тебе быть? — тест на профессию по типу личности

Одностраничное приложение (SPA) на чистом HTML/CSS/JS, без сборки и зависимостей.
Пользователь отвечает на 48 утверждений по 5-балльной шкале согласия, алгоритм
считает баллы по четырём шкалам MBTI и выдаёт один из 16 типов личности с топ-5
подходящих профессий.

## Запуск

Это статический сайт — никакой сборки не нужно. Любой из вариантов:

```bash
# Python
python3 -m http.server 8080

# Node
npx serve .
```

Затем открыть `http://localhost:8080`. Можно просто открыть `index.html` в браузере.

## Структура

| Файл           | Назначение                                                        |
|----------------|-------------------------------------------------------------------|
| `index.html`   | Каркас всех четырёх экранов (welcome → quiz → loading → result)    |
| `style.css`    | Токены дизайна, экраны, анимации, адаптив (mobile-first)           |
| `app.js`       | Логика: экраны, прохождение, подсчёт баллов, результат, шеринг     |
| `questions.js` | База из 48 вопросов с привязкой к шкалам EI / SN / TF / JP         |
| `results.js`   | 16 типов личности: описание, сильные/слабые стороны, профессии     |
| `assets/`      | Иконки и изображения                                              |

## Особенности

- **Профиль-спектр.** На экране результата показывается, насколько сильно вы
  склоняетесь к каждой из четырёх черт (анимированные шкалы с процентами).
- **Идентичность типа.** У каждого из 16 типов свой акцентный цвет и иконка,
  которые подсвечивают всю карточку результата.
- **Шеринг по ссылке.** Результат кодируется в URL (`#ENFP.78.66.71.59`), поэтому
  ссылка открывается сразу на готовом результате — удобно делиться в мессенджерах.
- **Анимации.** Плавная смена экранов, поэтапное появление карточки результата,
  счётчики процентов, шиммер заголовка. Всё уважает `prefers-reduced-motion`.
- **Доступность.** Семантическая разметка, ARIA-роли прогресс-бара, управление с
  клавиатуры (`1`–`5`, `←`).

## Как считается результат

Каждый вопрос «работает» в пользу одной буквы шкалы (`pole`). Согласие добавляет
баллы этой букве, несогласие — противоположной. По каждой из четырёх шкал
выбирается буква с большим числом баллов → собирается код из 4 букв (например,
`INTJ`) → из `results.js` берётся соответствующий тип. Процент по шкале — это доля
баллов доминирующей черты, она же показывается на профиле-спектре.

## Управление с клавиатуры

- `1`–`5` — выбор варианта ответа
- `←` — предыдущий вопрос
